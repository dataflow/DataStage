Submitting Data as a Data Package

To submit data as a Data Package, browse to the directory you wish to package in the web interface and select 'submit as data package'. The system will them guide you through the processes of adding metadata to your package, selecting a destination repository and submitting the data.

You will then be given options to: Submit the directory as a new Data Package or Add to a Data Package. Additionally, if the directory has previously been submitted as a Data Package, you will have the option to Update an existing package.

The significance of the three options are:

  1. Submit the directory as a new Data Package - create a new Data Package with the data contained within the directory you are submitting
  2. Add to a Data Package - add the directory and it's data into an existing Data Package
  3. Update an existing package - replace the data within an exisitng Data Package with the data within the current directory.
