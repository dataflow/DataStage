Data permissions and file organization

There are three levels of access to data within {{ settings.SITE_NAME }}: private, shared and collaborative:

  • private: data which can only be accessed by you
  • shared: data to be shared between research group members
  • collaborative: data to be shared with selected external collaborators (in addition to members of your research group)

Note: Research group leaders also have access to all data within a research group, including the 'private' files of individual group members.

Permissions within {{ settings.SITE_NAME }} are managed by file organisation; that is what permissions a particular set of data has is determined by where it is located.

  • data/ - base for research data files:
    • private/ - area for research group members private files:
      • username - private area for files belonging to username
    • shared/ - area for sharing files between research group members:
      • username - shared files created by username
    • collab/ - area for sharing files with selected external collaborators:
      • username - collaborator-shared files created by username
